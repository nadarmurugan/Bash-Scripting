# üñ• Linux System Monitoring Script (Bash + Cron)

## üìå Project Overview

This is a simple Bash-based system monitoring script that logs basic system performance metrics into a log file.

The script collects:

* üìä System Load Average (1, 5, 15 minutes)
* üß† Memory Usage
* üíæ Disk Usage (Root partition)
* üïí Timestamp for each entry

It is designed to be automated using **cron** for continuous monitoring.

## üìú Script Explanation

The script:

1. Defines a log file path.
2. Appends the current date and time.
3. Extracts:

   * Load average using `uptime`
   * Memory usage using `free -h`
   * Disk usage using `df -h /`
4. Appends all output to a log file.
5. Keeps historical data using append (`>>`).

# ‚è∞ Automating Using Cron (Every 2 Minutes)

## Step 1: Get Full Path of Script

## Step 2: Open Crontab

```bash

```

---

## Step 3: Add This Line

```bash

```

---

## üß† Cron Format Explanation

```
*/2  ‚Üí Every 2 minutes
*    ‚Üí Every hour
*    ‚Üí Every day of month
*    ‚Üí Every month
*    ‚Üí Every day of week
```

This runs the script every 2 minutes.
